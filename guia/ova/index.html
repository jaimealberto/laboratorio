OvA

  • Ubuntu Server

Muchas soluciones de software nos proveen ovas para poder realizar pruebas. La mayoría por no decir todas en idioma y teclado no es en Español, la zona horaria no es la nuestra, etc. La mayoría basadas en Ubuntu Server, según aparezcan mas actualizare esta documentación. Están son mis notas para poder cambiar todo esto:

Ubuntu Server

  • Zona horaria
timedatectl set-timezone Europe/Madrid && timedatectl
  • Idioma
dpkg-reconfigure locales
  • Mapa de teclado
dpkg-reconfigure keyboard-configuration
